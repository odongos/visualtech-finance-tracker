GitHub Sync Settings

Your data will be saved as visualtech-data.json in your GitHub repository. Every change syncs automatically.

The repo must exist and be accessible with your token. Can be private.
Needs repo scope. Stored only in your browser's localStorage — never sent anywhere except GitHub.
How to set this up
  1. Go to github.com/settings/tokens/new
  2. Give it a name (e.g. VisualTech Finance) and tick repo scope
  3. Click Generate token and copy it
  4. Create a new GitHub repository (can be private)
  5. Fill in the fields above and click Save & Enable Sync
